展場研究模組Gallery Study Modules

這一頁整理與 VR Gallery Spatial System 相關的兩個互動模組，讓觀眾從空間配置與光線變化兩個角度理解虛擬展場的觀看經驗。

This page gathers two interactive modules related to VR Gallery Spatial System, helping visitors understand virtual gallery viewing through spatial layout and changes in light.

觀看方式Viewing Mode

兩個模組都可直接在瀏覽器中觀看。它們目前作為網頁展覽中的互動內容，適合用來理解展場尺度、觀看距離與照明條件。

Both modules can be viewed directly in the browser. They currently function as interactive material within the web exhibitions, useful for understanding scale, viewing distance, and lighting conditions.

展場空間規劃器Gallery Spatial Planner

平面與立面同步的展場配置模組，可用來理解牆面位置、預留量體、觀看方向與通行淨空之間的關係。

A plan-and-elevation gallery layout module for understanding the relationship between wall placement, reserve volumes, viewing direction, and circulation clearance.

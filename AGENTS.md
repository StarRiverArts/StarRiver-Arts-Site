# StarRiverSite AGENTS

This project is the StarRiver Arts static site (GitHub Pages), located at `StarRiver-Arts-Site/`.

See root `../AGENTS.md` for workspace-wide safety rules and the full Windows CJK encoding standard.

## Windows CJK Encoding Guardrails

HTML, CSS, JS, and JSON files contain Chinese text. Apply before any PowerShell command involving Chinese content:

```powershell
chcp 65001 > $null
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
[Console]::InputEncoding  = [System.Text.Encoding]::UTF8
$OutputEncoding           = [System.Text.Encoding]::UTF8
```

- If terminal output shows `ä¸­æ–‡`, `????`, or `�` — discard it. Use the Read tool instead.
- Never use `Out-File` / `Set-Content` on HTML/JS/JSON files in PS 5.1 without `-Encoding utf8NoBOM`.
- After editing any file with Chinese content, verify with the Read tool before reporting done.

## Git Notes

- The site uses a git repository at `StarRiver-Arts-Site/`.
- The `play/` sub-directory is currently **untracked** — git cannot restore files there.
- Always check `git status` before and after edits.
- Do not force-push or reset without explicit user approval.

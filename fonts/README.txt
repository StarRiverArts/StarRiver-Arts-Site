Font files are not shipped in this design system. @font-face uses local() first and the woff2 files listed act as documentation. Google Fonts fallbacks: Barlow, Noto Sans TC, JetBrains Mono — load them via Google Fonts when using this system in HTML:

<link href="https://fonts.googleapis.com/css2?family=Barlow:wght@400;600;700;800;900&family=Noto+Sans+TC:wght@400;500;700;900&family=JetBrains+Mono:wght@400;600;800&display=swap" rel="stylesheet">

Voxel light-probe replacement. Required for the multi-light soft setups in this system — gives smooth per-pixel illumination for both static walls and avatars in one pass.
VR Stage Lighting 劇場光 free
Lighting FX
light
GPU-instanced "stage" lights with DMX/MIDI/AudioLink hooks. Use for the *projected* and *volumetric* virtual-light types — performant alternative to real-time Unity spots.

Default stack

Body architecture → Mochie's Standard. Stylised works → lilToon. World illumination → VRC Light Volumes. Stage / volumetric → VR Stage Lighting.

VR safety

Avoid screen-space volumetric fog (looks fake in stereo). Model light shafts as geometry + baked light instead — confirmed working in VR.

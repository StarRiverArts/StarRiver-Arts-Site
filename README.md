# StarRiverSite

The StarRiver Arts website — a static site published to GitHub Pages. The actual site lives in [`StarRiver-Arts-Site/`](StarRiver-Arts-Site/).

It is one site with four entry surfaces: `/studio` (portfolio), `/play` (VRChat worlds), `/museum` (exhibitions), and `/works` (creator universe), sharing a common nav, footer, and type system.

## Start here

- [`StarRiver-Arts-Site/ENTRY_POINTS.md`](StarRiver-Arts-Site/ENTRY_POINTS.md) — the four landing surfaces and their visual identity
- [`StarRiver-Arts-Site/SITE_ARCHITECTURE.md`](StarRiver-Arts-Site/SITE_ARCHITECTURE.md) — site structure
- [`StarRiver-Arts-Site/README.md`](StarRiver-Arts-Site/README.md) — site-level readme
- `StarRiver-Arts-Site/index.html` — site root (plus `studio.html`, `play.html`, `museum.html`)

## Notes

- Published via GitHub Pages (see `StarRiver-Arts-Site/.github/workflows/`).
- Interactive pages (RacingClub / TimeAttack) and the design-system tools live under `StarRiver-Arts-Site/play/` and `StarRiver-Arts-Site/museum/`.
- Content here is also editable through **StarRiverCMS** (`http://127.0.0.1:3737`), which targets this site as its `SITE_ROOT`.
